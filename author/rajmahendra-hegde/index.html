Rajmahendra Hegde

I have no special talents. I am only passionately curious

Raj has been working in Information Technology with special focus in the areas of Java development and Agile-Lean Project Management for over 20 years. Since the time he started his career as a Java developer has experienced has spanned several domains including ERP, EIP, Manufacturing, Healthcare. He worked in many areas of software development including designing, development, testing, and quality.

Raj is an Agile Practitioner by profession and a Java/Scala/Kotlin hacker and Community Guy at night!

Currently, he is working for JPMorgan as Technical Agile Coach in Hyderabad City.

Java Space Raj is a Community Guy since 2007, He has formed and leads two Java User Group (Chennai, Hyderabad). Currently, he is leading the Kotlin Hyderabad Community. He is a Java Champion and NetBeans DreamTeam member. He contributes to the Java community in both OpenSource and to the core team. He is an individual member at JCP and been an Expert Group member for some JSR.

Agile Practioner: Raj is a strong believer of the Agile and Lean way of project management. He has worked as Team Member, Scrum Master, and Product Owner. He has completed CSM and CSPO and pursuing more in the future on PMI-ACP, CSP, LeSS, etc.

Specialties: Java, Java EE, Java FX, JVM Languages ( Groovy, Scala, and Kotlin), Glassfish, NetBeans, OSGI, Continuous Integration ( Jenkins), Build tools (Ant, Maven, Gradle), Agile development, Scrum-Lean, XP, Kanban, LeSS, Clear Code
